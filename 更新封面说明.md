# 自动化更新视频封面

## 使用方法

### 1. 安装 Node.js
确保已安装 Node.js（https://nodejs.org/）

### 2. 运行脚本

在项目目录下打开终端，运行：

```bash
node update-video-covers.js
```

或者使用 npm script：

```bash
npm run update-covers
```

### 3. 脚本功能

- ✅ 自动读取 `index.html` 文件
- ✅ 提取所有 `data-bvid` 属性中的 BV 号
- ✅ 通过 B站 API 获取每个视频的封面 URL
- ✅ 自动添加 `data-cover` 属性到 HTML
- ✅ 自动备份原文件（.backup.时间戳）
- ✅ 显示更新进度和结果

### 4. 输出示例

```

提取 BV 号...

开始获取 17 个视频的封面...

  ✓ 成功: https://i0.hdslb.com/bfs/archive/xxx.jpg
[2/17] 正在获取 BV1iVzmB4ExX...
  ✓ 成功: https://i0.hdslb.com/bfs/archive/xxx.jpg

已为 17 个视频添加封面链接。
```

### 5. 注意事项

- 脚本会自动备份原文件，文件名格式：`index.html.backup.时间戳`
- 如果某个视频获取失败，会继续处理其他视频
- 请求间隔 500ms，避免请求过快被封 IP
- 如果遇到 403 错误，可能是 IP 被限制，可以稍后重试

### 6. 手动更新单个视频

如果某个视频获取失败，可以手动添加：

```html
<div class="video-preview-wrap" data-bvid="BV1uMrSBxEP3" data-cover="https://i0.hdslb.com/bfs/archive/xxx.jpg">
```

### 7. 重新运行

可以随时重新运行脚本，它会：
- 更新已有的 `data-cover` 属性
- 为新添加的视频添加封面
- 不会影响其他内容
